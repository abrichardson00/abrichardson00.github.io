About Me

I'm Andrew Richardson, a 4th year informatics student at the University of Edinburgh. I'm studying on the 5 year integrated Masters of Informatics degree programme. One can view my CV here.

Throughout my degree I've enjoyed working on larger software-engineering style projects, while also really enjoying more mathematical topics - data science, computer simulations etc... Recently, areas like quantum computing and computational neuroscience have been really interesting to study. While defenitely strongest with Python or Java, I've used C/C++ a fair amount and I love making fun things with Javascript.

I code for fun whenever I've got the time, and I've set up this website to briefly overview different projects I've worked on. This includes some university work I've found interesting. This website is a more brief and organised overview of my skills, but I also host code on Github.

I set up this website by adapting a free CSS template from http://freehtml5.co - but when I've got more time I plan to remake one from scratch and improve my web developement skills. I'll also add more projects, like my honours project I'm currently working on.
